CPU (Central Processing Unit)

หัวใจสำคัญของคอมพิวเตอร์ ทำหน้าที่ประมวลผลข้อมูลต่างๆ

  • CPU หรือหน่วยประมวลผลกลางเป็นส่วนประกอบฮาร์ดแวร์ที่เป็นหน่วยคำนวณหลักในเซิร์ฟเวอร์ เซิร์ฟเวอร์และอุปกรณ์อัจฉริยะอื่นๆ จะแปลงข้อมูลเป็นสัญญาณดิจิทัลและดำเนินการทางคณิตศาสตร์กับอุปกรณ์เหล่านั้น CPU เป็นองค์ประกอบหลักที่ประมวลผลสัญญาณและทำให้สามารถประมวลผลได้ ซึ่งทำหน้าที่เป็นสมองของอุปกรณ์ประมวลผลต่างๆ

GPU (Graphics Processing Unit)

สำคัญสำหรับการแสดงผลภาพกราฟิก

  • GPU คือโปรเซสเซอร์ที่ออกแบบมาเป็นพิเศษเพื่อจัดการกับงานการเรนเดอร์กราฟิกที่หนักหน่วง กราฟิกที่สร้างด้วยคอมพิวเตอร์ เช่นที่พบในวิดีโอเกมหรือสื่อภาพเคลื่อนไหวอื่นๆนั้นถูกกำหนดให้แต่ละเฟรมแยกจากกันเพื่อให้สามารถ 'วาด' ทีละเฟรมได้โดยใช้คอมพิวเตอร์ ซึ่งจะต้องใช้พลังงานจำนวนมากกว่างานกราฟฟิคทั่วไป
